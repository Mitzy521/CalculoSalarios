# CalculoSalarios
Implementé un sistema de nómina en C# que utiliza el polimorfismo mediante una función virtual para calcular los salarios de distintos tipos de empleados. Definí una clase base con un método virtual llamado CalcularSalario(), el cual contiene la lógica general, y luego creé clases derivadas que utilizan el modificador override para adaptar el cálculo según sus condiciones específicas, como bonos fijos o pagos extras. Toda la gestión se centraliza en una lista dinámica, lo que me permite recorrerla y ejecutar el método de cálculo, asegurando que cada objeto responda con su propio salario correcto gracias a la vinculación dinámica de métodos en tiempo de ejecución
